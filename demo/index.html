BP Calendar

A lightweight, dynamic calendar library with date range selection

Single Date Selection

Click on any date to select it. Past dates are disabled and shown in light grey.

Selected Date:

Code Example

new BP_Calendar('#calendar-single', {

  mode: 'single',

  monthsToShow: 2,

  onDateSelect: (date) => console.log(date)

});

Date Range Selection

Click a start date (highlighted in golden-yellow), then click an end date (also highlighted in orange). The range will be highlighted in light grey. Hover over dates to see a preview with duration tooltip. This example has a minimum of 2 days and maximum of 5 days in the range.

Selected Range:

Code Example

new BP_Calendar('#calendar-range', {

  mode: 'range',

  monthsToShow: 2,

  tooltipLabel: 'Nights',

  dateConfig: { 'YYYY-MM-DD': { date, isDisabled, price, minDays, maxDays } },

  defaultMinDays: 2,

  onRangeSelect: (range) => console.log(range)

});

Multiple Months Display

Display 1-4 months at once. This example shows 3 months.

Code Example

new BP_Calendar('#calendar-multi', {

  monthsToShow: 3,

  mode: 'single'

});

Advanced Features

This example demonstrates dateConfig: per-date isDisabled, price, minDays, maxDays.

Code Example

dateConfig: {

  '2026-01-10': { date: '2026-01-10', isDisabled: false, price: '$120', minDays: 2, maxDays: 5 },

  '2026-01-15': { date: '2026-01-15', isDisabled: true, price: null, minDays: null, maxDays: null },

}, defaultMinDays: 2

Datepicker Mode

Click on the input fields to open the calendar popup. Select a date range and the inputs will be populated automatically. The popup closes after selection or when clicking outside.

Selected Range:

Code Example

new BP_Calendar('#calendar-datepicker', {

  mode: 'datepicker', monthsToShow: 2, tooltipLabel: 'Nights',

  dateConfig: { ... }, defaultMinDays: 2,

  onRangeSelect: (range) => console.log(range)

});
